📋 Cara Deploy Apps Script (wajib jika URL berubah):
  1. Buka Google Sheet → Extensions → Apps Script
  2. Hapus semua kode lama → paste Code.gs terbaru
  3. Klik Deploy → New Deployment (bukan edit!)
  4. Type: Web app · Execute as: Me · Access: Anyone
  5. Copy URL → paste di bawah → klik Simpan & Test
